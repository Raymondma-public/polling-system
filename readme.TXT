Instructions

Start client(Angular)
1.Go to root of the project folder
2.Install node dependencies
	npm install
3.Go to app app client folder 
	cd .\polling-system-client\
4.Start the app client server
	npm start


Start api server(Node.js with express)
1.Go to root of the project folder
2.Install node dependencies
	npm install
3.Go to app api server folder 
	cd .\polling-system-server\
4.Start the api server
	npm start

Go to http://localhost:4200/ to access the page
